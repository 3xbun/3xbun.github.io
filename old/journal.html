อนุทินครั้งที่ 2

วันที่ 16 สิงหาคม พ.ศ. 2561

วันนี้อาจารย์สอนการใช้โปรแกรม Photoshop CC 2017 ตัดต่อรูปภาพ การวาง Layer การสร้างกรอบโดยใช้เครื่องมือ Polygonal Lasso tool และการสร้างเงา

การสร้างกรอบหน้างต่างนั้น
1. ให้เราสร้างตัวขนาดกรอบก่อนโดยใช้ Polygonal Lasso tool ถ้าต้องการให้เส้นของกรอบนั้นตรงให้กดShiftในขณะที่ลากเส้นนั้น หลังจากนั้นลากเส้นให้เป็นรูปกรอบหน้างต่าง 1 ด้าน

2. การทำให้บริเวณกรอบที่เราสร้างมีแสงเข้า การใช้เครื่องมือ Brightness ให้คลิกที่ Image จากนั้นคลิก Adjustments แล้วคลิกที่ Brightness/Contrast หลังจากนั้นเราก็คลิกที่ลูกศรตรง Brightness ไปทางขวาเพื่อเพิ่มแสงสว่างให้กับบริเวณที่เราทำกรอบเอาไว้หรือจะพิมพ์ตัวเลขตรงช่อง Brightness ก็ได้

จากการเรียนวันนี้ทำให้มีทักษะการเพิ่มแสงเฉพาะส่วน โดยใช้เครื่องมือ Polygonal Lasso tool อีกทั้งยังสามารถปรับใช้กับเครื่องมือ Rectangular Marquee Tool ได้และเพิ่มทักษะในการเลือกเฉพาะส่วนในการทำอย่างอื่นๆได้อีกด้วย

อนุทินครั้งที่ 3

วันที่ 30 สิงหาคม 2561

วันนี้ได้เรียนทฤษฎีเรื่องหลักการออกแบบโปสเตอร์ และการใช้ Photoshop CC 2017 ในการปรับภาพในมุมต่างๆ การใช้ Pen Tool การใส่เอฟเฟค และ การใช้ Horizontal Type Tool

หลักการออกแแบบโปสเตอร์

1. ความเป็นเอกภาพ เป็นการทำให้เนื้อหาและองค์ประกอบทุกส่วนนั้นมีความสัมพันธ์กัน
2. ความสมดุล แบ่งออกเป็น 2 ประเภท

2.1 ซ้าย ขวาเท่ากัน
 2.2 ซ้ายขวาไม่เท่ากัน หากจะทำไม่เท่ากันก็ต้องมีสิ่งที่หลอกจาให้มันดูสมดุลกัน เช่น การใช้สีดึงดูดสายตา

3. สัดส่วน การจัดวางองค์ประกอบต้องไม่เยอะ
4. จุดเด่น สร้างจุดสนใจให้ผู้ดูได้รับข้อมูลหลักตามที่ตั้งเป้าหมายไว้ การออกแบบโปสเตอร์ มีส่วนประกอบ คือ ภาพและข้อความ โดยข้อความแบ่งออกเป็น 3 หัวข้อย่อย ได้แก่

1. ขนาดของตัวอักษร ขนาดใหญ่ สำหรับพาดหัว และ ขนาดเล็ก สำหรับข้อความรายละเอียด
2. รูปแบบตัวอักษร เช่น ความสวยงาม ชัดเจน มีความสอดคล้องกับข้อความ
3. การกำหนดโครงสี สีที่ใช้กับตัวอักษรนั้น ไม่ควรมีมากเกินไป ระวังเรื่องคู่สีตรงข้ามจะใช้ในสัดส่วนเท่ากันไม่ได้ ต้องมีอันใดอันหนึ่งมากกว่า

การทำPhotoshop ได้เรียนเรื่อง

1. การปรับขนาดภาพและมุมมองภาพต่างๆ การปรับขนาดภาพให้กด Ctrl + T ก็จะมีสี่เหลี่ยมขึ้นมา และ มีแถบสำหรับปรับความกว้าง ความสูงและองศาอยู่ข้างบนรูปภาพ แต่ถ้าต้องการปรับรูปให้เป็นแบบมองจากมุมต่างๆ ให้คลิกขวา เช่นต้องการมุมที่เอียงจากด้านใน ให้ คลิกขวาและกด Skew ต้องการให้รูปดูนูน คลิกขวาและกด Warp
2. การใช้ Pen Tool ในการทำให้เกิดเส้นโค้ง และการเทสีในพื้นที่ ที่เราทำให้กดที่ Shapeโดยทำการเลือกสีก่อนกด
3. การใส่เอฟเฟคต่างๆ เช่นทำกรอบให้กับรูปที่เราเลือก หรือทรงรูปภาพที่เราทำใช้ Stroke ได้ทั้งข้างในและข้างนอกรูป 4.Horizontal Type Tool ได้เรียนรู้ทั้งการทำเอฟเฟค Warp Text การขยับตัวอักษรให้ห่าง ให้ชิดกัน ทั้งในบรรทัดและระหว่างบรรทัด

อนุทินครั้งที่ 4

วันที่ 6 กันยายน 2561

ได้เรียนการแต่งภาพการทำใบหน้าให้เนียน โดยอาจารย์เริ่มสอนตั้งแต่การลง Plug in Imagenomic โดยมี Filter Noiseware Portraiture RealGrain Noiseware ใช้เพื่อลบสิว และ Portraiture ทำให้หน้าเนียนมากขี้น

โดยในคาบอาจารย์ให้ใช้เครื่องมือ Portraiture โดยการใช้ ให้เข้าที่ Filter >> Imagenomic >> Portraiture และทำการปรับส่วนของผิวตามที่เราต้องการจะทำให้รูปภาพหน้าเนียน แล้วจุดที่เป็นสิว หรือรอยด่างดำนั้นจางลง และวันนี้อาจารย์ให้ชิ้นงานคือการทำกรอบรูป Facebook โดยใช้เครื่องมือ Select >> Modify >> Border เมื่อกดแล้วจะมีกำหนดความกว้างของกรอบ แล้วใช้ฟังก์ชัน Stroke ในการเพิ่มสี จากวันนี้ได้เรียนรู้การลบรอยบนใบหน้า หลักการทำ กรอบรูปโดยใช้ Border และใช้ฟังก์ชั่น Stroke และยังสามารถประยุกต์ความรู้ที่เคยเรียนมาจากคาบที่แล้วมาปรับใช้ในการออกแบบกรอบรูปได้

อนุทินครั้งที่ 6

วันที่ 20 กันยายน 2561

ในวันนี้อาจารย์สอนการลบพื้นหลังในพื้นที่ ที่เล็ก ๆ ลบได้ยาก โดยสอนด้วยกัน 2 วิธี

วิธีที่ 1 คือ Select รูปโดยใช้ Magic Wand Tool(เครื่องมือที่ 4 ทางซ้ายมือ หลังจากนั้นให้กด Edit in Quick Mask Mode (สัญลักษณ์สี่เหลี่ยมที่มีวงกลมตรงกลาง) แล้วรูปจะขึ้นสีแดง ต่อจากนั้นให้ใช้ยางลบลบ ถ้าลบเกินให้ใช้แปรงระบายส่วนที่เกิน

วิธีที่ 2 ใช้ Quick Selection Tool หลังจากนั้นให้กด Edit in Quick Mask Mode (สัญลักษณ์สี่เหลี่ยมที่มีวงกลมตรงกลาง) แล้วกด Select and Mask และกด Refine Edge Brush Tool

วันนี้อาจารย์ให้ทำชิ้นงาน Infographic โดยใช้เว็บไซต์ Piktochart เมื่อเข้าไปแล้ว ให้กด Create New และเลือก Infographic เมื่อเข้าไปแล้วจะมีTemplatesหรือเราจะเลือกเป็นแบบกระดาษว่างก็ได้ หลังจากนั้นทางด้านซ้ายจะมีฟังก์ชั้นให้แทรก รูปร่างต่างๆ จากการเรียนคาบนี้ได้เรียนรู้วิธีการลบพื้นที่เล็กๆโดยใช้เครื่องมือ Magic Wand Tool และ Quick Selection Tool และได้รู้จักเว็บไซต์ Piktochart ที่จะช่วยการออกแบบ Infographic และ Presentation

อนุทินครั้งที่ 7

วันที่ 4 ตุลาคม 2561

วันนี้อาจารย์ได้สอนเรื่องการใช้เครื่องมือ Brush และการใช้ Styleตัวอักษรและการลงเพิ่ม โดยเครื่องมือ Brushมีทั้งหมด 5 แบบ

1. คือ Spot Healing Brush Tool เป็นการลบจุดที่เราเลือกโดยเมื่อเราลบบริเวณพื้นหลังตรงที่เราลบแล้วพื้นหลังก็จะกลมกลืนกับพื้นหลังบริเวณอื่น

2. Healing Brush Tool เป็นการลบโดยเราต้องกด Alt แล้วคลิ๊กซ้ายบริเวณที่เราต้องการให้เป็นพื้นหลังตรงที่เราต้องการจะลบเพื่อให้แทนจุดที่เราลบออก

3. Patch Tool เป็นการลากเส้นบริเวณที่ต้องการลบออก หรือคัดลอก หากต้องการลบออกให้ลากเส้นเป็นบริเวณที่ต้องการลบออกแล้วเลือก Sourceจากแถบข้างบน และลากกรอบที่เราคลุมบริเวณที่ต้องการลบจากตรงที่ต้องการลบออกไปตรงที่พื้นหลังที่เราต้องการแทนที่ ถ้าต้องการคัดลอกบริเวณที่เราเลือกจากตรง Source ให้เปลี่ยนเป็น Destination แล้วลากเส้นกรอบบริเวณที่เราต้องการคัดลอกแล้วก็ลากไปบริเวณที่เราต้องการที่คัดลอกอยู่

4. Content-Aware Move Tool ให้ลากเส้นไปบริเวณที่เราต้องการสลับพื้นที่กัน โดยเมื่อลากเล้นแล้วให้ลากไปที่บริเวณที่เราต้องการสลับ

5. Red Eye Tool ช่วยในการแก้ไขรูปที่ถ่ายแล้วคนในรูปใส่คอนแทคเลนส์ แล้วมีตาแดงขึ้นมาให้สามารถลบออกได้

การเปลี่ยนพื้นหลังอักษรในรูปแบบ Styles โดยหากไม่มีรูปแบบ Styles ให้กดที่ Window แล้วเลือก Styles ก็จะมีแถบเครื่องมือ Styles ขึ้นมา แล้วมีรูปแบบให้เลือก พอเลือกรูปแบบแล้ว สามารถเปลี่ยนสีและรูปแบบเงาต่างๆได้โดยการคลิ๊กที่เอฟเฟค โดยจะอยู่มุมขวาล่างสัญลักษณ์ fx หากต้องการรูปแบบ Style เพิ่มเติมสามารถดาวน์โหลดได้ที่เว็บ www.brusheezy.com แล้วพิมพ์ค้นหาว่า Style แล้วทำการดาวน์โหลดรูปแบบที่ต้องการ เมื่อดาวน์โหลดแล้วให้แตกไฟล์ แล้วทำการคัดลอกไฟล์ที่เป็นรูป fx แลtทำการคัดลอกไปที่ File Adobe Photoshop แล้ว paste ไปที่ไฟล์ที่ชื่อว่า Style เพื่อให้อยู่ถาวร

อนุทินครั้งที่ 8

วันที่ 11 ตุลาคม 2561

วันนี้อาจารย์ได้สอนการใช้โปรแกรม Illustrator ซึ่งการใช้เครื่องมือมีความคล้ายคลึงกับโปรแกรม Photoshop เช่นการสร้างวงกลม ถ้าต้องการให้เป็นลักษณะกลมให้กด Shift ค้างไว้ในขณะสร้างซึ่งเหมือนกับการสร้างวงกลมในโปรแกรมPhotoshop ซึ่งในคาบนี้อาจารย์ได้สอนการสร้างรูปทรงเรขาคณิต โดยใช้เครื่องมือ

1. Ellipse Tool ในการสร้างวงกลม โดยถ้าต้องการให้กลมให้กด Shift ข้างในขณะที่สร้าง

2. Polygon Tool ใช้ในการสร้างรูปหลายเหลี่ยม ซึ่งสามารถปรับขยายหรือลดเฉพาะด้านได้โดยใช้เครื่องมือลูกศรที่มีพื้นหลังสีขาว(Direct Selection Tool) กด Double ไปที่เส้นที่เราต้องการปรับก็จะมีจุดขึ้นมาแล้วให้กดที่จุดๆนั้นแล้วลากออกไปเพื่อให้เกิดเป็นเส้นขอบที่ขยายออกหรือเข้าตามที่เราต้องการ และสามารถปรับจำนวนเหลี่ยมหรือทำรูปทรงให้เป็นวงกลมโดยเมื่อใช้เครื่องมือนี้ขณะที่สร้างให้กดปุ่มลูกศรขึ้นหรือลงที่คีย์บอร์ดค้างไว้เพื่อปรับให้ได้เหลี่ยมที่ต้องการ หรือเป็นทรงกลม (ต้องกดลูกศรขึ้น)

4. Star Tool ใช้ในการสร้างรูปดาว หากต้องการให้ดาวมีแฉกเพิ่มขึ้นให้กดลูกศรขึ้นที่คีย์บอร์ด แต่ถ้าต้อกงารลดแฉกลง ให้กดลูกศรลง สามารถปรับขยายหรือลดเฉพาะด้านได้โดยใช้วิธีเดียวกันกับตอนปรับรูปหลายเหลี่ยม และอาจารย์สอนได้การลงสีโดย การลงสีแบบปกติ การใช้ Library ในการลงพื้นหลังในรูปที่สร้าง การไล่สีโดยใช้เครื่องมือ Gradient Tool และ การให้เส้นในหลายๆรูปเชื่อมกันโดยใช้เครื่องมือPathFinder ซึ่งการคลุมรูปภาพหลาย ๆ รูปให้ใช้เครื่องลูกศรที่พื้นหลังว่าง (Selection Tool)

อาจารย์ได้สอนเกี่ยวกับการใส่สีในรูปที่สร้างขึ้นโดยจะมีการลงสีแบบธรรมดา และ ปรับเป็นสีแบบ gradient สอนการทำขอบโดยให้เส้นเป็นเส้นปะ การปรับรูปร่างของรูปดาว หรือ รูปที่มีเหลี่ยมโดยใช้ลูกศรขึ้นลงในการเพิ่ม ลด แฉกของดาว หรือเหลี่ยม และยังมีการทำเส้นโค้งโดยใช้ pen tool กดลากเส้นแล้วใช้ลูกศรสีขาวในการปรับให้เป็นเส้นโค้ง อาจารย์ได้ให้ทำรูปดาวที่มีตา และปาก ทำดอกไม้ ทำวงกลมที่มีเส้นปะในแบบต่างๆ โดยใช้วิธีที่ได้เรียนไป
